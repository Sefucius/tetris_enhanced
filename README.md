# 🎮 俄罗斯方块 - 增强版

一个功能丰富、视觉精美的俄罗斯方块游戏，使用 Python 和 Pygame 开发。

![Python](https://img.shields.io/badge/Python-3.6+-blue.svg)
![Pygame](https://img.shields.io/badge/Pygame-2.0+-green.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

## ✨ 特性

### 🎨 主题系统
- **6个独特的主题**，每个主题都有独特的配色方案、音乐风格和背景特效
- **随机主题加载** - 每次启动游戏体验不同的主题
- **即时主题切换** - 在设置菜单中随时切换主题
- **主题包括**：
  - 🌆 霓虹城市 - 赛博朋克风格
  - 🚀 太空科幻 - 深邃星空
  - 🎮 复古像素 - 8-bit怀旧
  - 🌊 海洋世界 - 动态波浪
  - 🌅 日落黄昏 - 温暖小调
  - 🌲 森林秘境 - 极光效果

### 🎵 动态音乐系统
- **程序生成音乐** - 实时合成背景音乐
- **多种调式** - 大调（明亮）和小调（忧郁）
- **不同风格** - 电子乐、复古、宁静
- **速度自适应** - 每个主题有独特的音乐速度

### 🎯 游戏功能
- **连击系统** - 连续消除获得额外分数
- **幽灵方块** - 显示方块落地预览
- **霓虹模式** - 炫酷的发光效果
- **统计数据** - 详细的游戏数据分析
- **排行榜** - 记录您的最佳成绩
- **成就系统** - 解锁各种成就

### 🎨 视觉效果
- **粒子效果** - 方块落地、行消除的华丽特效
- **屏幕震动** - 多行消除时的视觉冲击
- **冲击波** - 强力消除的震撼效果
- **浮动文字** - 分数、连击的动态显示
- **背景特效** - 渐变、星空、波浪、极光

## 📋 系统要求

- **操作系统**: Windows 10/11
- **Python**: 3.6 或更高版本
- **Pygame**: 2.0 或更高版本
- **内存**: 100MB 以上
- **存储**: 10MB 可用空间

## 🚀 安装

### 1. 克隆仓库

```bash
git clone https://github.com/你的用户名/tetris_enhanced.git
cd tetris_enhanced
```

### 2. 安装依赖

```bash
pip install pygame
```

## 🎮 如何运行

```bash
python tetris_enhanced.py
```

## 🕹️ 游戏控制

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↑ | 旋转方块 |
| ↓ | 加速下落 |
| 空格键 | 直接落地 |
| P | 暂停/继续 |
| R | 重新开始（切换主题） |
| N | 切换霓虹模式 |
| M | 静音/取消静音 |
| ESC | 打开设置菜单 |
| TAB | 查看统计面板 |
| H | 查看成就面板 |
| Q | 退出游戏 |

## ⚙️ 设置

在设置菜单（ESC）中可以调整：

- **音效开关** - 开启/关闭游戏音效
- **背景音乐** - 开启/关闭背景音乐
- **音乐音量** - 调整音乐音量（0-100%）
- **音效音量** - 调整音效音量（0-100%）
- **幽灵方块** - 显示/隐藏落地预览
- **霓虹模式** - 开启/关闭霓虹发光效果
- **主题选择** - 选择6个主题之一

## 📊 游戏机制

### 评分系统

- **单行消除**: 100 分 × 等级
- **双行消除**: 300 分 × 等级
- **三行消除**: 500 分 × 等级
- **四行消除 (Tetris)**: 800 分 × 等级
- **连击奖励**: 额外加分

### 等级系统

- 每消除 **10行** 升一级
- 等级越高，下落速度越快
- 速度公式: `500 - (level - 1) * 50` 毫秒

## 🏆 成就系统

游戏包含多种成就，例如：

- 🎯 **初出茅庐** - 完成一局游戏
- 💯 **完美发挥** - 达到等级5
- 🌟 **四连专家** - 消除4行
- 🔥 **连击大师** - 达到5连击
- 🏃 **马拉松** - 游戏时间超过10分钟
