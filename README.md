✨ 游戏特性

主题系统
  - 6个独特的主题，每个主题都有独特的配色、音乐和背景
  - 随机主题加载 - 每次启动游戏体验不同的主题
  - 即时主题切换 - 在设置菜单中随时切换主题

  主题包括：
    . 霓虹城市 - 赛博朋克风格
    . 太空科幻 - 深邃星空
    . 复古像素 - 8-bit怀旧
    . 海洋世界 - 动态波浪
    . 日落黄昏 - 温暖小调
    . 森林秘境 - 极光效果

动态音乐系统
  - 程序生成音乐 - 实时合成背景音乐
  - 多种调式 - 大调（明亮）和小调（忧郁）
  - 不同风格 - 电子乐、复古、宁静
  - 速度自适应 - 每个主题有独特的音乐速度

游戏功能
  - 连击系统 - 连续消除获得额外分数
  - 幽灵方块 - 显示方块落地预览
  - 霓虹模式 - 炫酷的发光效果
  - 统计数据 - 详细的游戏数据分析
  - 排行榜 - 记录您的最佳成绩
  - 成就系统 - 解锁各种成就

🚀 快速开始

方式一：从源代码运行

1. 安装 Python 3.6+
2. 安装 Pygame: pip install pygame
3. 运行游戏: python tetris_enhanced.py

方式二：使用可执行文件

1. 双击 Tetris Enhanced.exe
2. 按空格键或回车键开始
3. 使用方向键控制方块

🕹️ 游戏控制

基础操作
  ← →     左右移动
  ↑       旋转方块
  ↓       加速下落
  空格    直接落地

系统功能
  ESC     设置菜单
  P       暂停/继续
  R       重新开始（切换主题）
  N       切换霓虹模式
  M       静音/取消静音
  TAB     查看统计面板
  H       查看成就面板
  Q       退出游戏

⚙️ 设置与功能

设置菜单（ESC键）

开关设置：
  . 音效开关 - 开启/关闭游戏音效
  . 背景音乐 - 开启/关闭背景音乐
  . 幽灵方块 - 显示/隐藏落地预览
  . 霓虹模式 - 开启/关闭霓虹发光效果

音量与主题：
  . 音乐音量 - 调整音乐音量（0-100%）
  . 音效音量 - 调整音效音量（0-100%）
  . 主题选择 - 选择6个主题之一

其他功能：
  . 键位绑定（按K键）- 自定义所有按键
  . 恢复数据（底部按钮）- 恢复出厂设置

📊 游戏机制

评分系统
  单行消除: 100 × 等级
  双行消除: 300 × 等级
  三行消除: 500 × 等级
  四行消除: 800 × 等级
  连击奖励: 额外加分

等级系统
  - 每消除10行升一级
  - 等级越高，下落速度越快
  - 速度公式: 500 - (level - 1) × 50 毫秒

连击奖励
  2连击: 额外50%分数
  3连击: 额外100%分数
  4连击及以上: 额外150%分数

🎯 游戏技巧

基础技巧
  1. 规划落点 - 使用幽灵方块预览最佳位置
  2. 保持平整 - 避免形成空洞
  3. 优先消除 - 及时消除行，避免堆积过高
  4. 学会旋转 - 熟练掌握7种方块的旋转特性

高级技巧
  1. 连击策略 - 累积多行后一次性消除
  2. T-Spin - 使用T方块旋转消除（高难度）
  3. 保持冷静 - 速度加快后不要慌张
  4. 使用硬降 - 空格键瞬间落地，节省时间

❓ 常见问题

Q: 背景音乐不播放？
A: 在设置中开启"背景音乐"开关，检查音量设置

Q: 游戏运行缓慢？
A: 关闭霓虹模式（按N键），降低窗口大小

Q: 如何切换主题？
A: 按R重新开始会自动切换，或在设置菜单中选择

Q: 如何重置所有数据？
A: 在设置菜单点击"恢复所有数据到出厂设置"

Q: 统计数据丢失？
A: 检查tetris_statistics.json文件是否存在，数据无法恢复

🏗️ 构建可执行文件（开发者）

快速构建

  一键构建（Windows）:
    build.bat

  或使用Python脚本:
    python build_pyinstaller.py
    python build_release.py

系统要求

  开发环境:
    - Python 3.6+
    - PyInstaller: pip install pyinstaller
    - 操作系统: Windows 10/11

  运行环境（玩家）:
    - 操作系统: Windows 10/11
    - 不需要: Python或任何其他依赖

构建步骤

  1. 安装构建工具: pip install pyinstaller
  2. 构建可执行文件: python build_pyinstaller.py
  3. 测试可执行文件: 双击 dist/Tetris Enhanced.exe
  4. 创建发布包: python build_release.py

构建结果

  dist/Tetris Enhanced.exe (26-27 MB)
  release/Tetris_Enhanced_v1.0.0.zip - 发布包
